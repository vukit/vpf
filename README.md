# Уличная кормушка для собак крупных пород


Надежная автоматизированная система кормления, спроектированная специально для крупных пород собак и эксплуатации в открытых вольерах. Проект успешно прошел проверку реальными условиями (снег, дождь, жара) и находится в эксплуатации с 2024 года.

## Ключевые особенности

*   **Компактный механизм:** Вместо громоздкого шнека использован оригинальный узел с пропеллером, что позволило существенно уменьшить габариты корпуса без потери производительности.
*   **Всепогодность:** Кормушка предназначена для установки на открытых площадках. Не требует навеса, дополнительной защиты или регулярной чистки механизма.
*   **Безопасность (5В):** Устройство питается от напряжения 5 вольт. Это гарантирует полную электрическую безопасность для питомца даже в случае повреждения кабеля.
*   **Автономность и объем:** Резервуар вмещает до 3.5 кг корма, чего достаточно для обеспечения собаки едой в течение 7 дней.
*   **Умная логика выдачи:** При временном отключении питания устройство автоматически «догоняет» график, выдавая пропущенные порции сразу после возобновления работы.

## Технические характеристики

*   **Точность дозирования:** ±20 грамм (порции задаются в граммах).
*   **График кормления:** До 3-х сессий в сутки в заданное время.
*   **Управление:** Мобильное приложение (связь через BLE).
*   **Функции приложения:** 
    *   Настройка расписания.
    *   Выдача порции вне очереди по команде.
    *   Обновление прошивки (OTA firmware update).

## Аппаратная часть
* **Микроконтроллер:** ESP32-C3.
* **Двигатель:** Bringsmart JS-50B(S) BLDC, DC 5V, CW/CCW, 2rpm, 10kgf/cm².
* **RTC:** Модуль DS3231 I2C.
* **Измерение веса:** TZT HX711 весовой модуль, 4 датчика FGHGF 34X34mm 5kg.
* **Питание:** 5В.

## Программное обеспечение
* **Язык:** firmware - C (ESP-IDF, FreeRTOS), software - Java.

## Опыт эксплуатации
Кормушка работает на открытом воздухе более 2-х лет, не требуя специального укрытия или частого технического обслуживания. Конструкция доказала свою устойчивость к обледенению и забиванию механизмов.
